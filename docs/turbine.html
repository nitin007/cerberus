Turbine is a replacement for Engine. Unlike Engine, Turbine exports the class, not an instance of the class. Also, each Turbine request is a new request, only using data that is explicity provided. This helps eliminate "cross talk" of objects and cookies.

Turbine will follow redirects and automatically switch between HTTP and HTTPS. It does not check that a secure certificate is valid, so MITM attacks are possible.
